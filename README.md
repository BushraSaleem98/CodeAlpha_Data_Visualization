# CodeAlpha_Data_Visualization

This project is part of the CodeAlpha Internship and focuses on Data Visualization using the Netflix Titles Dataset.  
The goal is to transform raw data into meaningful visuals that reveal key insights about Netflix’s growth, content trends, and audience focus.

---

## Project Overview
The dataset contains detailed information about movies and TV shows available on Netflix, including title, country, release year, rating, genre, and more.  
This project uses Python (Matplotlib & Seaborn) to create insightful visualizations and uncover hidden patterns in Netflix’s content strategy.

---

## Dataset Information
- Dataset name: netflix_titles.csv  
- Source: [Netflix Shows Dataset on Kaggle](https://www.kaggle.com/datasets/shivamb/netflix-shows)  
- Shape: ~8,800 rows × 12 columns  
- Main columns: type, title, country, release_year, rating, listed_in, date_added

---

## Key Visualizations & Insights

| Visualization | Description | Key Insights |
|----------------|-------------|---------------|
| Content Growth Over Time | Bar chart showing number of titles added from 2008–2021 | Netflix’s content library grew rapidly after 2015, peaking around 2019–2020 |
| Movies vs TV Shows | Comparison of total movies vs TV shows | Movies dominate (~2:1 ratio), but TV shows are increasing steadily |
| Top 10 Content-Producing Countries | Horizontal bar chart of top countries | USA leads, followed by India and the UK |
| Top 10 Most Common Genres | Most frequent genres on Netflix | International Movies, Comedies, and Dramas are most popular |
| Content Ratings Distribution | Bar chart of ratings | TV-MA and TV-14 dominate, showing focus on mature audiences |

---

## Technologies Used
- Python 3  
- Pandas – Data manipulation  
- Matplotlib – Static visualizations  
- Seaborn – Stylish statistical plots  

---

## How to Run
1. Clone the repository:  
   `git clone https://github.com/BushraSaleem98/CodeAlpha_Data_Visualization`  
2. Navigate to the folder:  
   `cd CodeAlpha_Data_Visualization`  
3. Install dependencies:  
   `pip install pandas matplotlib seaborn`  
4. Run the script:  
   `python main.py`

---

## Sample Output
Each section of the code generates a clear visualization that highlights trends in Netflix’s global library — from yearly content growth to top genres and countries.

---

## Summary of Insights
- Netflix experienced exponential growth after 2015.  
- Movies still outnumber TV shows, but original series are on the rise.  
- The U.S. and India dominate Netflix’s content catalog.  
- Drama, Comedy, and International titles are most frequent.  
- Mature ratings (TV-MA, TV-14) are most common, showing audience focus.
